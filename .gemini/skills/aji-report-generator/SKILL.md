# Aji Report Generator Skill

This skill automates the creation of a 7-page PowerPoint report using data from the Ajinomoto Game Excel reports.

## Main Workflow
1. Update Excel: Run scripts/update_graphs.py
2. Generate PowerPoint: Run scripts/generate_pptx.py

## Key Features
- Preserves Template Style
- Dynamic Chart Sizing
- Trend Analysis
- Text Replacement
